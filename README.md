# bajins-java-clazz

- JDK原生API使用，不添加任何依赖包（包括J2EE、javax、sun）
- 设计模式、排序算法、数据结构等原生实现

* JDK源码 [https://github.com/openjdk](https://github.com/openjdk)


**JDK1.8主要依赖源码**

- `javafx-src.zip`
- `src.zip`

- `lib\src.zip`
- `legal`
- `jmods`
